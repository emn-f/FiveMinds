# FiveMinds App

Este é um protótipo de interface de usuário para o aplicativo FiveMinds, desenvolvido com HTML e CSS puros. O projeto inclui as telas de login, dashboard (com grupos e encontros), busca e chat.

## Estrutura do Projeto

A estrutura de arquivos do projeto é a seguinte:

```
/fiveminds
├── assets/
│   └── images/
│       └── fiveminds-logo.png  # Logo do aplicativo
├── css/
│   └── style.css               # Folha de estilo principal
├── js/
│   └── app.js                  # Scripts JavaScript
├── busca.html                  # Tela de Busca de Grupos/Encontros
├── dash.html                   # Dashboard principal
├── grupo.html                  # Tela de Chat de Grupo
├── index.html                  # Tela de Login
└── README.md
```

## Como Visualizar

Para visualizar o projeto, siga os passos abaixo:

1.  **Clone o repositório** (se estiver em um Git) ou **baixe os arquivos** do projeto.
2.  Navegue até a pasta raiz do projeto (`fiveminds-app/`).
3.  Abra os arquivos `login.html`, `index.html`, `busca.html` ou `grupo.html` diretamente no seu navegador de preferência (Google Chrome, Firefox, Edge, etc.).

    * Clique com o botão direito no arquivo HTML e selecione "Abrir com" -> "Seu navegador".

## Telas Implementadas

* **`login.html`**: Página de acesso ao aplicativo.
* **`index.html`**: Dashboard principal com "Meus Grupos" e "Próximos Encontros".
* **`busca.html`**: Tela para buscar novos grupos ou atividades.
* **`grupo.html`**: Exemplo de tela de chat dentro de um grupo.

## Responsividade

O projeto foi desenvolvido com foco em responsividade, adaptando-se a diferentes tamanhos de tela (desktop e mobile).

* Redimensione a janela do navegador para ver as mudanças no layout.
* Utilize as ferramentas de desenvolvedor do seu navegador (F12) e a "Visualização de Dispositivo" para testar em tamanhos específicos de celulares e tablets.

## Tecnologias Utilizadas

* **HTML5**: Estrutura e conteúdo das páginas.
* **CSS3**: Estilização e responsividade.
* **Font Awesome**: Ícones (assumindo que o CDN está incluído nos arquivos HTML).
